# DESIGN.md

## 气质与意象
Notepad++ 中文官网——清新绿意，如同晨露中的编程工作台。绿色（青蛙图标）是品牌灵魂，搭配白色卡片和浅灰背景，营造专业、轻量、高效的工具感。

## 配色方案
- 主色（品牌绿）：#90BE6D → 渐变 #6AB04C，意象来源：Notepad++ 青蛙图标
- 强调色：#27AE60（深绿按钮/链接）
- 背景白：#FFFFFF
- 区块浅灰背景：#F8FAFB
- 文字主色：#1F2937
- 文字次色：#6B7280
- 卡片边框：#E5E7EB
- 渐变 Hero：从 #6AB04C 到 #90BE6D 再到浅绿

## 字体排版
- 字体族：-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans SC", sans-serif
- 标题：700 bold，H1 2.5rem，H2 2rem，H3 1.5rem
- 正文：400，1rem，行高 1.75
- 代码块：monospace

## 页面结构
- 顶部导航：固定白色背景，左侧 Logo+名称，右侧导航链接（首页/下载/功能/版本/教程）
- Hero 区域：绿色渐变背景，大标题 + 描述 + CTA 按钮
- 内容区域：白色背景 + 浅灰交替分区，最大宽度 1200px 居中
- 卡片：白色圆角卡片（border-radius: 12px），微阴影（0 2px 12px rgba(0,0,0,0.06)）
- 底部 Footer：深色背景（#1F2937），品牌信息 + 链接

## 动效与交互
- 按钮 hover：轻微上移 + 阴影增强
- 卡片 hover：阴影加深
- 导航链接：下划线渐显
- 页面切换：无动画，保持快速响应
- Tab 切换：平滑过渡

## 设计禁忌
- 不要使用蓝紫渐变（与 Notepad++ 品牌不符）
- 不要使用过重的阴影
- 不要使用卡通/花哨装饰元素
- 不要使用纯黑背景（Footer 除外）
